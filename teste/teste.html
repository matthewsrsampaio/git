Códigos Usados

Adicionar o nome de usuário GLOBAL: git config --global user.name "SeuNome"

Adicionar o email GLOBAL: git config --global user.email "seuemail@seuemail.com"

Adicionar o nome de usuário LOCAL: git config user.name "SeuNome"

Adicionar o email LOCAL: git config user.email "seuemail@seuemail.com"

Verificar informações globais do GIT: git config --global --list

Verificar informações locai do GIT: git config --list

Deletar o repositório local: rm -rf .git

Verificar se ainda existe arquivos e pastas escondidas: ls -lah

Iniciar um repositório: git init

Clonar um repositório: git clone endereçoHTMLdoRepositório.git

Verificar o status do repositório: git status

Tirar um arquivo do STAGE, mas mantendo a configuração feita: git reset nomeDoArquivo

Manter um arquivo no STAGE, mas apagando as modificações feitas: git restore nomeDoArquivo

Ver a diferença dos arquivos em STAGE para os que não estão: git diff

Para sair de END: q

Commitar os arquivos em STAGE: git commit -m "mensagem explanatória sobre o que commir realizado"

Commitar os arquivos em STAGE adicionando algo que esteja fora do STAGE: git commit -a -m "mensagem explanatória sobre o que commir realizado"

Fazer um ajuste de última hora em um arquivo ja commitado: git add . e depois digita: git commit --amend -m "mensagem explanatória"
